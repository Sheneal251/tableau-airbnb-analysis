# Airbnb Tableau Dashboard Project

This project analyzes Airbnb listing data using Tableau to identify pricing trends, neighborhood performance, and host activity.

## Tools Used
- Tableau Desktop
- Excel / CSV Dataset

## Key Insights
- Pricing distribution by neighborhood
- Room type performance
- Host listing activity
- Availability trends

## How To View
Download the .twbx file and open it using Tableau Desktop or Tableau Reader.
